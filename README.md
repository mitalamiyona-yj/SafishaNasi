# SafishaNasi Cleaning Services Web Application
VIDEO LINK https://youtu.be/CQbOnWwYLB8 

## Project Description
SafishaNasi is a web application that connects customers with cleaning service providers. 
Users can request services such as home cleaning, office cleaning, and school cleaning.

## Features
- View available cleaning services
- Book cleaning services
- Simple and responsive user interface
- Backend API for managing services

## Technologies Used
- Node.js
- Express.js
- HTML
- CSS
- JavaScript
- MongoDB

## Installation

1. Clone the repository

git clone https://github.com/mitalamiyona-yj/SafishaNasi.git

2. Go to project folder

cd SafishaNasi

3. Install dependencies

npm install

4. Start the server

node server.js
